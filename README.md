# 2-days-bootcamp-java

Este repositório reúne os exercícios e projetos do bootcamp de Java de 2 dias.

Aqui estão alguns dos temas que estou estudando:

- Conceitos básicos de Java: tipos, variáveis e estruturas de controle
- Programação orientada a objetos: classes, objetos, herança e encapsulamento
- Estruturas condicionais e laços de repetição
- Uso de `lambda` e expressões funcionais simples
- Criação de jogos simples e lógica de interação entre personagens

Os arquivos estão organizados em duas pastas principais:

- `game/` – exemplos de um jogo simples com heróis e monstros
- `JavaCore/` – exercícios dos fundamentos da linguagem Java
